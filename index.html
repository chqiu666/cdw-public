2D Spatial Canvas（p5.js）

练习意图：以基础图元与网格为语法，探索2D空间构成与动态更新；关注简洁几何、色彩与层次关系。 交互要点：持续绘制、基本图元组合。数据：内置参数，无外部数据集。

3D Spatial Canvas（Three.js）

练习意图：通过基本几何体、光照与轨道相机，建立最小三维空间构图原型。 交互要点：旋转、轨道控制、网格参考。数据：内置参数，无外部数据集。

Temporal Structure（D3.js）

练习意图：以时间轴/甘特与日历热图表达时序关系与强度分布，强调比例、分组与标注的可读性。 交互要点：悬浮提示、并发区间、刻度与图例。 数据：本地 CSV（如 events.csv、synthetic-data.csv），在 GitHub Pages 上可直接加载。

Relational Structure（D3.js Force）

练习意图：用力导引布局与多变量编码展示实体关系；尝试标签、连线、方向与拖拽交互。 交互要点：拖拽节点、悬浮高亮、缩放平移。 数据：CSV（nodes.csv 与 edges.csv）。

Geospatial Structure（Mapbox GL JS）

练习意图：以点/线/面图层与数据驱动样式表达地理要素与语义；扩展到外部 GeoJSON 加载与视域拟合。 交互要点：悬停强调、弹窗、筛选与键盘快捷。 数据：内置几何与本地 manhattan.geojson。 注意：Mapbox 访问令牌已在示例脚本内配置，仅作课程展示用途。

练习意图：整合常见输入控件，建立人机交互的可用性基线；联动输出区即时反馈用户状态。 交互要点：滑条、复选、单选、下拉、文本输入的事件监听与状态汇总。 数据：表单状态（即时生成）。
